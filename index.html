# About HLS & DASH

Both HLS and DASH are http-based adaptive bitrate video transport protocols. HLS was developed by Apple in the early 2010s, as a mobile alternative flash-based rtmp streaming. MPEG-DASH is a more recent, open source alternative, for which native browser support is growing every day. Both are used by companies like Netflix, YouTube and Hulu to deliver streaming media content to their millions of customers every day.

They both work very similar, instead of providing the browser with a reference to a video file, a reference to a playlist, containing various video files of various sizes, is provided. The browser will then select the most appropriate file for the user's current available bandwidth.

Both HLS and DASH are supported by all modern browsers through a player like video js (which is integrated with the plugin).
