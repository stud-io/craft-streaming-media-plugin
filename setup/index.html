# Setup

In this chapter we will walk you through the steps required to setup the plugin, before media files can be transcoded and used in your entries.

# Install

# Using Craft Plugin Marketplace

Search for "Streaming Media"

# Using Composer

composer install stud-io/craft-streaming-media-plugin

# Configure Backends

The plugin uses a concept of "Backends" to transcode and store media files. Before Stream Assets can be added, you will need to configure one or more Backends. Several types of Backends are available and can be used.
